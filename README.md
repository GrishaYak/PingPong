# Ping Pong 
This is a usual ping pong game, except it is not. It is going to have more interesting physics and nice-looking interface.

In main branch there are only the most stable version, to watch new versions that are not fully ready yet, check other branches.

Code structure
--
This project is working in pair with my other one called [MonoGameLib](https://github.com/GrishaYak/MonoGameLib). MonoGameLib is some kind of engine that contains classes that are to be used in many different games.

Game is working using MonoGame framework.
